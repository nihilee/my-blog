# Modern Theme for Cryogen

A modern, responsive theme for the Cryogen blog framework.

## Features

- Modern design with clean layout
- Responsive layout for all devices
- Dark/Light mode toggle
- Inter font family
- Smooth animations and transitions
- Easy to customize

## Installation

1. Copy the `modern` folder to your `themes` directory
2. Update `content/config.edn` to set `:theme "modern"`

## Development

```bash
# Start development server
lein serve

# Build site
lein run
```

## Customization

- **Colors**: Edit CSS variables in `css/style.css`
- **Layout**: Modify HTML templates in `html/`
- **Navigation**: Update `base.html` for menu changes
- **JavaScript**: Add interactions in `js/script.js`
